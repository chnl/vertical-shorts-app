## Generate Prisma Client

Generate Prisma client

```
npx prisma generate
```

Migrate dev

```
npx prisma migrate --dev

```

## Deploy Remix Client via Netlify CLI

### Deploy Preview

From root directory run:

```
netlify deploy --build

```

### Deploy to Production

From root directory run:

```
 netlify deploy --build --prod
```
